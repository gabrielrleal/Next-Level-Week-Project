

##  HAPPY

Happy is an application that connects people to institutional care homes to make many children's day happier.


## Tech/framework used

Built in:

- [Node.js](https://nodejs.org/en/)
- [React](https://reactjs.org)
- [TypeScript](https://www.typescriptlang.org/)
